# Gatekeeper AI

Gatekeeper AI is a Java Spring Boot application that integrates with AI services to provide advanced features for your projects.

## Features
- AI service integration
- Webhook support
- Configurable via `application.properties`

## Requirements
- Java 17 or higher
- Maven 3.6+

## Getting Started
1. Clone the repository:
   ```bash
   git clone (https://github.com/codezmr/GateKeeperAI.git)
   ```
2. Build the project:
   ```bash
   mvn clean install
   ```
3. Run the application:
   ```bash
   mvn spring-boot:run
   ```

## Project Structure
- `src/main/java/com/gatekeeper/api/` - Main application code
- `src/main/resources/` - Configuration files
- `libs/` - External libraries
